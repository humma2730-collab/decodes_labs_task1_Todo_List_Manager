# 📋 To-Do List Manager

A professional command-line To-Do List Manager built with Python. This project allows users to manage daily tasks efficiently using a simple menu-driven interface. All tasks are stored permanently in a JSON file.

## 🚀 Features

- ✅ Add New Task
- 👀 View All Tasks
- ✏️ Update Existing Task
- 🗑️ Delete Task
- ✔️ Mark Task as Completed
- 🔍 Search Task
- 📊 View Task Statistics
- 💾 JSON File Storage
- 🆔 Automatic Task ID Generation
- 🕒 Created & Updated Timestamps
- ⚠️ Input Validation
- 🛡️ Error Handling
- 📂 Modular Programming
